# Autosignplus

idk what to put here
Features:
- Multiple named presets with cycle modes (FIXED/SEQUENTIAL/RANDOM)
- Color codes via & prefix
- Templates: {player}, {n}, {date}, {time}, {x}/{y}/{z}, {server}, {dim}
- Sneak-bypass to edit signs normally
- HUD overlay
- In-game config GUI
- Persistent config at config/autosign.json

Default keybinds:
- Y: toggle autosign
- O: open config GUI
- I: cycle mode
